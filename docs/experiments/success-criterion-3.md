# 成功基準3 実務者テスト — 実験記録(雛形)

- **成功基準3 (01-intent.md)**: 現役設計者が2〜10部品の実在ブラケットAssyをADCで
  記述し、「従来CADの構想設計より速い、または知識が残る点で優る」と評価すること
- **形式**: 被験者(現役設計者)がClaude Code+ADC(examples/starter-repo構成)で
  実在部品を設計する。進行役は原則介入しない(質問には答えるが操作しない)
